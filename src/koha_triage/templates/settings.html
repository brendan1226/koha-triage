GitHub integration

Optional: a GitHub token lets the code generator fetch file contents from the Koha GitHub mirror. The token only needs public_repo read scope (raises rate limits from 60/hr to 5000/hr).

Patch workflow

Koha does not accept pull requests. Instead:

  1. Generate a code fix on any bug detail page
  2. Download the .patch file
  3. Apply locally: git am bug_XXXXX.patch
  4. Review, test, and adjust as needed
  5. Attach to Bugzilla: git bz attach XXXXX HEAD
